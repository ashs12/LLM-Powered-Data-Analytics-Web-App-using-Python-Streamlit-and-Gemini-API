AI-Powered Data Analytics Agent

A web-based data tool that allows users to upload CSV files and receive instant insights using the **Google Gemini 2.5 Flash model. 

Features
Intelligent Analysis:** Ask natural language questions about your data.
Automated Visualization:** Generates charts (Matplotlib/Seaborn) based on data trends.
Cloud Deployed:** Hosted on Streamlit Cloud for global access.

Tech Stack
Language: Python
API: Google Gemini API
Frontend:** Streamlit
Data Exploration and Explanation: Pandas, Matplotlib, Seaborn

How to Use
1. Upload any `.csv` file.
2. Type a question like "Summarize the sales trends for last month".
3. View the AI-generated response and corresponding charts.
